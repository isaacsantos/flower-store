# Tech Stack

## Core
- React 19 (JSX, functional components, hooks)
- Vite 8 (build tool + dev server)
- Plain CSS (co-located per component, no CSS-in-JS or utility frameworks)

## Linting
- ESLint 9 with `eslint-plugin-react-hooks` and `eslint-plugin-react-refresh`

## Fonts
- Playfair Display (headings) and Lato (body) via Google Fonts

## Common Commands

```bash
# Run dev server
npm run dev

# Production build
npm run build

# Preview production build
npm run preview

# Lint
npm run lint
```

All commands should be run from the `flower-store/` directory.
